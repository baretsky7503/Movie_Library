# Movie Library - Личная кинотека

**Автор:** Анатолий Ким

## Описание

Десктопное приложение для ведения личной коллекции фильмов с графическим интерфейсом. Позволяет добавлять, удалять, фильтровать и сохранять фильмы в JSON-файл.

## Функционал

- Добавление фильмов с указанием названия, жанра, года выпуска и рейтинга
- Отображение всех фильмов в таблице
- Удаление выбранных фильмов
- Фильтрация по жанру (частичное совпадение)
- Фильтрация по году выпуска (точное совпадение)
- Автоматическое сохранение данных в `movies.json`
- Валидация ввода:
  - Название и жанр не могут быть пустыми
  - Год — целое число от 1888 до 2026
  - Рейтинг — число от 0 до 10
